迭代器

c++提供以下几种迭代器供使用：

  1. begin/cbegin：返回指向起始的迭代器
  2. end/cend：返回指向末尾的迭代器
  3. rbegin/crbegin：返回指向起始的逆向迭代器
  4. rend/crbegin：返回指向末尾的逆向迭代器

需要注意的是，上面所列举的迭代器前面的c表示const
