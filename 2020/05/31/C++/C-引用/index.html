引用的概念

引用是变量的一个别名，相当于已经存在的变量的另一个名字。引用名与被引用的变量名都指向同一个内存地址。

  2. 引用是始终如一的，即引用了一个变量后就不能再引用其它变量
  3. 引用和被引用的变量都指向同一个内存地址
  4. 引用只能引用同一种类型的变量，不能引用常量。int a=10;double &b=a;这种引用是非法的，以及int &a=10;也是非法的。
