在实际的编程中，我们可能需要对输出格式进行控制，比如以某进制输出一个数，或者控制浮点数的精度，再或者控制输出的位宽。这是就需要用到流操纵算子进行格式控制。

控制浮点数精度

控制浮点数精度可以用到以下两个方法：

  • precision：cout的成员函数，调用的方法为：cout.precision(5)
  • setprecision：流操纵算子，调用方式为：cout<<setprecision(5)

上面两个方法功能相同，

使用非定点方式输出时，指定输出浮点数的有效位数。

定点方式输出时，指定输出浮点数小数点后面的有效位数。

定点方式：小数点必须出现在个位数后面

设置域宽的流操纵算子

设置域宽可以使用以下两个方法：

  • setw：流操纵算子，调用方法为：cin>>setw(4)或cout<<setw(4)
  • width：成员函数，调用方法为：cin.windth(4)或cout.width(4)
